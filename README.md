## cwj_co_net

一个基于网络事件驱动的协程调度框架，主要用于快捷的搭建各种服务器，包括但不限于tcpserver,httpserver,rpc框架；

### 项目特点

- 利用ucontext实现有栈协程，以网络事件为驱动，对协程进行调度
- 利用hook技术把非阻塞IO操作包装成同步操作，减轻用户编写和调试压力
- 提供丰富的工具模块，以便被一般的项目复用
- 仅支持linux平台
